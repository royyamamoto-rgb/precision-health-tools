# Precision Health Calculator Suite

A free, comprehensive health calculator web app built with React, Vite, and Tailwind CSS.

## Features

### Calculators Included

1. **BMI (Body Mass Index)** - Calculate and categorize body mass index
2. **BMR (Basal Metabolic Rate)** - Mifflin-St Jeor equation
3. **TDEE (Total Daily Energy Expenditure)** - Maintenance calories
4. **Goal Calories** - Cutting, bulking, or maintenance targets
5. **Macro Calculator** - Daily protein, carbs, and fat in grams
6. **Weight Loss Timeline** - Estimate time to reach goal weight

## Tech Stack

- React 19 + Vite 7
- Tailwind CSS 4
- Deployed on Vercel

## Quick Start

```bash
npm install
npm run dev      # Development server
npm run build    # Production build
npm run preview  # Preview build
```
